# Data Directory

This directory contains source texts and generated output for the Open Bible Translation project.

## Directory Structure

```
data/
├── sources/           # Source text files (not committed to git)
│   ├── hebrew/        # Hebrew OT source texts
│   ├── greek/         # Greek NT source texts
│   └── *.manifest.json # Dataset manifests
└── output/            # Generated pipeline output (not committed)
    ├── tokens/        # Tokenized source data
    ├── translations/  # Generated translations
    ├── alignments/    # Token-to-span alignments
    ├── notes/         # Study notes
    └── index/         # Search indexes
```

## Source Data

Source texts must be downloaded separately due to licensing considerations. The pipeline expects:

### Hebrew Old Testament

**Recommended Source**: [Tanach.us Open Hebrew Bible](https://tanach.us)
- License: Public Domain
- Format: WLC (Westminster Leningrad Codex) with cantillation

### Greek New Testament

**Recommended Source**: [SBLGNT](https://sblgnt.com)
- License: CC BY 4.0
- Format: Plain Greek text with verse markers

## Dataset Manifests

Each source must have a `.manifest.json` file describing:
- License information
- Attribution requirements
- Content type
- Books covered

Example manifest:
```json
{
  "id": "tanach-wlc",
  "name": "Westminster Leningrad Codex",
  "version": "4.20",
  "license": {
    "name": "Public Domain",
    "summary": "No copyright restrictions"
  },
  "attribution": "Westminster Leningrad Codex from Tanach.us",
  "accessDate": "2024-01-15",
  "contentType": "source-text",
  "languages": ["hebrew"],
  "books": ["Gen", "Exod", "..."]
}
```

## Generated Output

The `output/` directory is generated by the pipeline and should not be committed to git.

To regenerate output:
```bash
pnpm pipeline:full
```

## License Compliance

**IMPORTANT**: Never commit copyrighted content to this repository.

Before adding any source data:
1. Verify the license permits redistribution
2. Create a manifest documenting the license
3. Add proper attribution

If unsure about a license, do not include the data.
